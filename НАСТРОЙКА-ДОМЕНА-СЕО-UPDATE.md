# 🌐 НАСТРОЙКА ДОМЕНА seo-update.ru

## ⚡ БЫСТРАЯ ИНСТРУКЦИЯ:

### Шаг 1: Добавить домен в Vercel

1. Откройте: https://vercel.com/dashboard
2. Найдите проект `seo-services-mvp`
3. Перейдите: **Settings** → **Domains**
4. Нажмите: **"Add Domain"**
5. Введите: `seo-update.ru`
6. Нажмите: **"Add"**

### Шаг 2: Настроить DNS записи

Vercel покажет вам DNS записи, которые нужно добавить у регистратора домена.

**Обычно это:**
- **A запись**: `@` → IP адрес Vercel
- **CNAME запись**: `www` → `cname.vercel-dns.com`

**Или:**
- **CNAME запись**: `@` → `cname.vercel-dns.com`
- **CNAME запись**: `www` → `cname.vercel-dns.com`

### Шаг 3: Добавить DNS записи у регистратора

1. Войдите в панель управления доменом (где покупали домен)
2. Найдите раздел "DNS" или "Управление DNS"
3. Добавьте записи, которые показал Vercel
4. Сохраните изменения

### Шаг 4: Обновить переменные окружения

1. В Vercel Dashboard → **Settings** → **Environment Variables**
2. Найдите или добавьте: `SITE_URL`
3. Установите значение: `https://seo-update.ru`
4. Нажмите **"Save"**
5. Появится уведомление → нажмите **"Redeploy"**

### Шаг 5: Дождаться распространения DNS

- Обычно: 1-2 часа
- Максимум: 24-48 часов
- Проверить можно: https://dnschecker.org (введите `seo-update.ru`)

### Шаг 6: Проверить работу

После распространения DNS:
1. Откройте: `https://seo-update.ru`
2. Должен открыться ваш сайт
3. Проверьте SSL (должен быть зеленый замочек)

## 🔍 ПРОВЕРКА DNS:

Проверить, распространились ли DNS записи:
1. Откройте: https://dnschecker.org
2. Введите домен: `seo-update.ru`
3. Выберите тип записи: `A` или `CNAME`
4. Нажмите "Search"
5. Если записи распространились - увидите зеленые галочки по всему миру

## ⚠️ ВАЖНО:

1. **DNS может занять до 48 часов** - наберитесь терпения
2. **После добавления DNS** - подождите 1-2 часа
3. **После обновления SITE_URL** - обязательно сделайте Redeploy
4. **SSL сертификат** выдается автоматически Vercel (может занять до 1 часа)

## 🆘 ЕСЛИ НЕ РАБОТАЕТ:

1. **Проверьте DNS записи:**
   - Убедитесь, что они добавлены правильно
   - Проверьте через dnschecker.org

2. **Проверьте в Vercel:**
   - Settings → Domains → должен быть статус "Valid Configuration"
   - Если ошибка - исправьте DNS записи

3. **Проверьте переменные окружения:**
   - SITE_URL должен быть `https://seo-update.ru`
   - После изменения - сделайте Redeploy

---
