# ⚡ БЫСТРАЯ НАСТРОЙКА ДОМЕНА seo-update.ru

## 🎯 ЧТО НУЖНО СДЕЛАТЬ:

### 1. В Vercel Dashboard:

1. Откройте ваш проект в Vercel
2. Перейдите: **Settings** → **Domains** (или вкладка **"Domains"**)
3. Нажмите: **"Add Domain"**
4. Введите: `seo-update.ru`
5. Нажмите: **"Add"**

### 2. Vercel покажет DNS записи:

Скопируйте DNS записи, которые покажет Vercel. Обычно это:
- **A запись** для `@` → IP адрес
- **CNAME** для `www` → `cname.vercel-dns.com`

### 3. У регистратора домена:

1. Войдите в панель управления доменом (где покупали seo-update.ru)
2. Найдите раздел **"DNS"** или **"Управление DNS"**
3. Добавьте записи, которые показал Vercel
4. Сохраните

### 4. Подождите 1-2 часа:

DNS записи распространяются 1-24 часа. Проверить можно: https://dnschecker.org

### 5. Обновите SITE_URL:

1. Vercel Dashboard → **Settings** → **Environment Variables**
2. Найдите/добавьте: `SITE_URL`
3. Установите: `https://seo-update.ru`
4. Сохраните → **Redeploy**

---

## ✅ ГОТОВО!

После этого `https://seo-update.ru` будет работать!
