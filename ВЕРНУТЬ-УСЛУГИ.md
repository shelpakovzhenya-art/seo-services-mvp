### Способ 2: Через seed скрипт (АВТОМАТИЧЕСКИ)

**Если у вас установлен Vercel CLI:**

```bash
# 1. Установите Vercel CLI
npm i -g vercel

# 2. Войдите
vercel login

# 3. Подключитесь к проекту
vercel link

# 4. Получите переменные окружения
vercel env pull .env.local

# 5. Заполните базу данных (создаст все услуги)
npx tsx prisma/seed.ts
```

**После выполнения:**
- Все 11 услуг будут созданы автоматически
- Они появятся на главной странице
- Они появятся на странице /services
- Они будут в калькуляторе
