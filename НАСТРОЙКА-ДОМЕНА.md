## 🔧 Настройка домена в Vercel:

### Шаг 1: Добавить домен в Vercel

1. Откройте проект в [Vercel Dashboard](https://vercel.com/dashboard)
2. Перейдите в **Settings** → **Domains**
3. Нажмите **Add Domain**
4. Введите: `seo-update.ru`
5. Нажмите **Add**

### Шаг 2: Настроить DNS записи

Vercel покажет вам DNS записи, которые нужно добавить у вашего регистратора домена:

**Обычно это:**
- **A запись**: `@` → IP адрес Vercel
- **CNAME запись**: `www` → `cname.vercel-dns.com`

**Или:**
- **CNAME запись**: `@` → `cname.vercel-dns.com`
- **CNAME запись**: `www` → `cname.vercel-dns.com`

### Шаг 3: Настроить переменные окружения

В Vercel Dashboard → Settings → Environment Variables добавьте:

```
SITE_URL=https://seo-update.ru
```

**Важно:** Используйте `https://` (не `http://`)

### Шаг 4: Дождаться распространения DNS

DNS изменения могут занять от нескольких минут до 48 часов. Обычно это занимает 1-2 часа.

### Шаг 5: Проверить SSL сертификат

Vercel автоматически выдаст SSL сертификат для вашего домена. Проверьте:
- `https://seo-update.ru` работает
- Сертификат валидный (зеленый замочек в браузере)

## 🔍 Проверка после настройки:

### 1. Проверка доступности:
```bash
curl -I https://seo-update.ru
```

Должен вернуть статус `200 OK`

### 2. Проверка редиректов:
- `http://seo-update.ru` → `https://seo-update.ru` (автоматически)
- `https://www.seo-update.ru` → `https://seo-update.ru` (настроено в next.config.js)

### 3. Проверка SEO файлов:
- `https://seo-update.ru/sitemap.xml` - должен отображаться
- `https://seo-update.ru/robots.txt` - должен отображаться

### 4. Проверка метаданных:
Откройте любую страницу и проверьте в исходном коде:
- `<title>` правильный
- `<meta name="description">` правильный
- `<link rel="canonical">` правильный
- Open Graph теги присутствуют

## 📝 Обновление robots.txt и sitemap.xml:

После настройки домена обновите файлы через админ-панель:

1. Войдите в админ-панель: `https://seo-update.ru/admin`
2. Перейдите в **SEO файлы**
3. Обновите `robots.txt`:
   ```
   User-agent: *
   Allow: /
   
   Sitemap: https://seo-update.ru/sitemap.xml
   ```
4. Обновите `sitemap.xml` с правильными URL
