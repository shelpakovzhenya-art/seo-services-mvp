# 🚨 ПРОБЛЕМА: DATABASE_URL не настроен!

## ❌ Ошибка:
```
Error validating datasource `db`: the URL must start with the protocol `postgresql://` or `postgres://`
```

## ✅ РЕШЕНИЕ:

### Шаг 1: Настроить DATABASE_URL в Vercel

1. Откройте: https://vercel.com/dashboard
2. Найдите проект → **Settings** → **Environment Variables**
3. Найдите или добавьте переменную: `DATABASE_URL`
4. Значение должно быть в формате:
   ```
   postgresql://user:password@host:port/database
   ```
   
   **Пример:**
   ```
   postgresql://postgres:password123@db.example.com:5432/mydb
   ```

### Шаг 2: Проверить формат DATABASE_URL

**Правильный формат:**
- ✅ `postgresql://user:password@host:port/database`
- ✅ `postgres://user:password@host:port/database`

**Неправильный формат:**
- ❌ `file:./prisma/dev.db` (это для SQLite, не для продакшена)
- ❌ Пустая строка
- ❌ Без протокола

### Шаг 3: После добавления DATABASE_URL

1. **Сохраните** переменную
2. **Сделайте Redeploy** (появится уведомление)
3. **Дождитесь** завершения деплоя
4. **Попробуйте войти** в админку

## 🔍 ГДЕ ВЗЯТЬ DATABASE_URL:

### Если используете Vercel Postgres:

1. Vercel Dashboard → ваш проект → **Storage** → **Postgres**
2. Создайте базу данных (если еще нет)
3. Скопируйте **Connection String**
4. Добавьте его как `DATABASE_URL` в Environment Variables

### Если используете внешний PostgreSQL:

1. Получите строку подключения от вашего провайдера БД
2. Формат: `postgresql://user:password@host:port/database`
3. Добавьте в Vercel Environment Variables

## ⚠️ ВАЖНО:

1. **DATABASE_URL обязателен** для работы сайта
2. **Формат должен быть правильным** - начинаться с `postgresql://` или `postgres://`
3. **После изменения** - обязательно сделайте Redeploy

## 🆘 ЕСЛИ НЕТ БАЗЫ ДАННЫХ:

### Вариант 1: Создать Vercel Postgres (бесплатно)

1. Vercel Dashboard → ваш проект → **Storage** → **Postgres**
2. Нажмите **"Create"**
3. Выберите план (можно бесплатный)
4. Скопируйте Connection String
5. Добавьте как `DATABASE_URL`

### Вариант 2: Использовать внешний PostgreSQL

- Timeweb, Beget, REG.RU и другие предоставляют PostgreSQL
- Получите строку подключения
- Добавьте в Vercel
