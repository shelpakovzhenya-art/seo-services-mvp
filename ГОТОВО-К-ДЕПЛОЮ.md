## ⚠️ ВАЖНО: После деплоя настройте базу данных

SQLite **НЕ работает** на Vercel. Нужна PostgreSQL.

### Быстрое решение - Supabase (5 минут):

1. Зайдите на https://supabase.com
2. Создайте проект (бесплатно)
3. Скопируйте **Connection String** (URI)
4. В Vercel → Settings → Environment Variables:
   - Добавьте `DATABASE_URL` = ваш PostgreSQL Connection String
5. Передеплойте проект

### Обновите Prisma схему:

В `prisma/schema.prisma` измените:

```prisma
datasource db {
  provider = "postgresql"  // вместо "sqlite"
  url      = env("DATABASE_URL")
}
```

Закоммитьте и запушьте это изменение тоже.
