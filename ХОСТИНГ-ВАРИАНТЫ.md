## 📋 Пошаговая инструкция для Vercel (бесплатно):

### Шаг 1: Подготовка проекта
1. Убедитесь, что проект загружен на GitHub
2. Проверьте, что `.env` не в репозитории (он в `.gitignore`)

### Шаг 2: Деплой на Vercel
1. Зайдите на https://vercel.com
2. Войдите через GitHub
3. Нажмите **"Add New Project"**
4. Выберите репозиторий `my-site`
5. Vercel автоматически определит Next.js
6. Нажмите **"Deploy"**

### Шаг 3: Настройка переменных окружения
В настройках проекта добавьте:
- `DATABASE_URL` - для продакшена используйте внешнюю БД
- `ADMIN_USER` и `ADMIN_PASS`
- `SMTP_*` - настройки почты
- `SITE_URL` - URL вашего сайта

### Шаг 4: Подключение домена
1. В настройках проекта → **Domains**
2. Добавьте свой домен
3. Следуйте инструкциям по настройке DNS

## 📋 Пошаговая инструкция для VPS (Timeweb):

### Шаг 1: Покупка VPS
1. Зайдите на https://timeweb.com/ru/services/vps
2. Выберите тариф от 199₽/мес
3. Оплатите

### Шаг 2: Настройка сервера
1. Получите доступ к серверу (SSH)
2. Установите Node.js:
   ```bash
   curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
   sudo apt-get install -y nodejs
   ```

### Шаг 3: Установка проекта
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/shelpakovzhenya-art/my-site.git
   cd my-site
   npm install
   ```

### Шаг 4: Настройка PM2 (для автозапуска)
```bash
npm install -g pm2
npm run build
pm2 start npm --name "my-site" -- start
pm2 save
pm2 startup
```

### Шаг 5: Настройка Nginx (для домена)
```bash
sudo apt install nginx
# Настройте конфиг Nginx для проксирования на порт 3000
```
