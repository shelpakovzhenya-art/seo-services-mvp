# 🚨 СРОЧНО: Исправить вход в админку

## Проблема:
Не могу войти в админку - пишет "неверные учетные данные"

## ✅ РЕШЕНИЕ (выберите один способ):

---

## Способ 1: Через Vercel CLI (РЕКОМЕНДУЕТСЯ)

```bash
# 1. Установите Vercel CLI (если еще не установлен)
npm i -g vercel

# 2. Войдите в Vercel
vercel login

# 3. Подключитесь к проекту
vercel link

# 4. Получите переменные окружения
vercel env pull .env.local

# 5. Установите DATABASE_URL (скопируйте из .env.local)
# Откройте .env.local и скопируйте DATABASE_URL

# 6. Установите переменную окружения
export DATABASE_URL="ваша_строка_из_env_local"

# 7. Исправьте админа
npm run admin:fix

# Или напрямую:
node scripts/fix-admin-production.js
```

---

## Способ 2: Через Vercel Dashboard (если нет CLI)

### Вариант A: Через Vercel Functions

1. Откройте Vercel Dashboard → ваш проект
2. Перейдите в **Settings** → **Functions**
3. Создайте новую функцию или используйте существующую
4. Или используйте Vercel CLI (способ 1)

### Вариант B: Через Environment Variables + Redeploy

1. В Vercel Dashboard → **Settings** → **Environment Variables**
2. Убедитесь, что есть:
   ```
   ADMIN_USER=admin
   ADMIN_PASS=admin123
   DATABASE_URL=postgresql://...
   ```
3. Если нет - добавьте их
4. Сделайте **Redeploy** (это запустит seed, который создаст админа)

## Способ 3: Через Prisma Studio (визуально)

```bash
# 1. Получите переменные
vercel env pull .env.local

# 2. Установите DATABASE_URL
export DATABASE_URL="ваша_строка_из_env_local"

# 3. Откройте Prisma Studio
npm run prisma:studio

# 4. Откройте таблицу "Admin"
# 5. Проверьте, есть ли запись с username "admin"
# 6. Если нет - создайте новую запись:
#    - username: admin
#    - password: $2a$10$... (хеш пароля)
# 7. Или удалите старую и создайте новую
```

**Чтобы получить хеш пароля:**
```bash
node -e "const bcrypt = require('bcryptjs'); bcrypt.hash('admin123', 10).then(h => console.log(h))"
```

## 🆘 ЕСЛИ НИЧЕГО НЕ ПОМОГЛО:

1. **Проверьте переменные окружения в Vercel:**
   - ADMIN_USER должен быть `admin`
   - ADMIN_PASS должен быть `admin123`
   - DATABASE_URL должен быть правильным

2. **Проверьте подключение к базе данных:**
   - Убедитесь, что DATABASE_URL правильный
   - Проверьте, что база данных доступна

3. **Проверьте логи Vercel:**
   - Dashboard → Deployments → последний деплой → Logs
   - Ищите ошибки при создании админа

4. **Попробуйте создать админа вручную через SQL:**
   ```sql
   -- Подключитесь к вашей PostgreSQL базе
   -- Выполните:
   
   INSERT INTO "Admin" (id, username, password, "createdAt", "updatedAt")
   VALUES (
     'admin-1',
     'admin',
     '$2a$10$rKxZxZxZxZxZxZxZxZxZxZxZxZxZxZxZxZxZxZxZxZxZxZxZxZxZx',
     NOW(),
     NOW()
   )
   ON CONFLICT (username) DO UPDATE
   SET password = EXCLUDED.password;
   ```

   **Чтобы получить правильный хеш:**
   ```bash
   node -e "const bcrypt = require('bcryptjs'); bcrypt.hash('admin123', 10).then(h => console.log(h))"
   ```
