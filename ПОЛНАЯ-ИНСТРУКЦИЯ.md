## 🌐 ШАГ 2: НАСТРОИТЬ ДОМЕН seo-update.ru

### 1. Добавить домен в Vercel

1. Откройте: https://vercel.com/dashboard
2. Проект → **Settings** → **Domains**
3. Нажмите: **"Add Domain"**
4. Введите: `seo-update.ru`
5. Нажмите: **"Add"**

### 2. Настроить DNS записи

Vercel покажет DNS записи. Добавьте их у регистратора домена:

**Обычно это:**
- **A запись**: `@` → IP адрес
- **CNAME**: `www` → `cname.vercel-dns.com`

### 3. Обновить SITE_URL

1. Vercel Dashboard → **Settings** → **Environment Variables**
2. Найдите/добавьте: `SITE_URL`
3. Установите: `https://seo-update.ru`
4. Нажмите **"Save"**
5. Нажмите **"Redeploy"** в уведомлении

### 4. Дождаться DNS (1-24 часа)

Проверить можно: https://dnschecker.org

## 🆘 ЕСЛИ НЕ РАБОТАЕТ:

### Услуги не появились:
1. Проверьте админку → Услуги → все ли услуги созданы
2. Проверьте, что услуги активны (isActive = true)
3. Выполните seed скрипт еще раз

### Домен не работает:
1. Проверьте DNS записи через dnschecker.org
2. Проверьте в Vercel → Domains → статус должен быть "Valid"
3. Подождите еще 1-2 часа (DNS может занять время)
