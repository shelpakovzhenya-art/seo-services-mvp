## ✅ РЕШЕНИЕ (делайте по порядку):

### 1. НАСТРОЙТЕ ДОМЕН seo-update.ru

**В Vercel Dashboard:**

1. Откройте: https://vercel.com/dashboard
2. Найдите проект → **Settings** → **Domains**
3. Нажмите **"Add Domain"**
4. Введите: `seo-update.ru`
5. Vercel покажет DNS записи - добавьте их у регистратора домена
6. Дождитесь распространения DNS (1-24 часа)

**Пока домен не настроен, используйте:** `https://seo-services-mvp-5yfr.vercel.app`

### 2. ЗАПОЛНИТЕ БАЗУ ДАННЫХ (чтобы разделы работали)

**Способ 1: Через Vercel CLI (БЫСТРО)**

```bash
# Установите Vercel CLI
npm i -g vercel

# Войдите
vercel login

# Подключитесь к проекту
vercel link

# Получите переменные окружения
vercel env pull .env.local

# Заполните базу данных
npx tsx prisma/seed.ts
```

**Способ 2: Через админ-панель (ПРОЩЕ)**

1. Откройте: https://seo-services-mvp-5yfr.vercel.app/admin/login
2. Войдите: `admin` / `admin123`
3. Перейдите: **Настройки** → **Меню**
4. Убедитесь, что все пункты меню созданы и активны

### 3. НАСТРОЙТЕ ПЕРЕМЕННЫЕ ОКРУЖЕНИЯ

**В Vercel Dashboard → Settings → Environment Variables:**

Добавьте/проверьте:
```
DATABASE_URL=postgresql://ваша_строка_подключения
SITE_URL=https://seo-update.ru
ADMIN_USER=admin
ADMIN_PASS=admin123
```

**ВАЖНО:** После изменения → **Redeploy** последний деплой!

1. **Проверьте логи Vercel:**
   - Dashboard → Deployments → последний деплой → Logs

2. **Проверьте базу данных:**
   - Убедитесь, что DATABASE_URL правильный
   - Проверьте, что база доступна

3. **Проверьте консоль браузера:**
   - F12 → Console → ищите ошибки
