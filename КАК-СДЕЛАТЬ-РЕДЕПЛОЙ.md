# 🔄 Как сделать Redeploy в Vercel

## Способ 1: Через Deployments (основной способ)

1. Откройте [Vercel Dashboard](https://vercel.com/dashboard)
2. Найдите ваш проект `seo-services-mvp`
3. Перейдите в раздел **"Deployments"** (слева в меню)
4. Найдите последний деплой (самый верхний)
5. Наведите курсор на деплой или нажмите на него
6. Появится меню с тремя точками **"..."** справа
7. Нажмите на **"..."** → выберите **"Redeploy"**

**Или:**
- Нажмите на сам деплой (на его название/коммит)
- Откроется страница деплоя
- В правом верхнем углу будет кнопка **"Redeploy"**

## Способ 2: Через Settings (после изменения переменных)

1. Откройте проект в Vercel Dashboard
2. Перейдите в **Settings** → **Environment Variables**
3. Добавьте/измените переменные
4. Нажмите **"Save"**
5. Появится уведомление: **"Redeploy to apply changes?"**
6. Нажмите **"Redeploy"** в уведомлении

## Способ 3: Через Git (самый простой)

Сделайте пустой коммит и запушьте:

```bash
git commit --allow-empty -m "Force redeploy"
git push
```

Vercel автоматически задеплоит новую версию.

## Способ 4: Через Vercel CLI

```bash
# Установите Vercel CLI (если еще не установлен)
npm i -g vercel

# Войдите
vercel login

# Подключитесь к проекту
vercel link

# Сделайте редеплой
vercel --prod
```

## Способ 5: Через кнопку "Deploy" (если есть)

1. В Vercel Dashboard → ваш проект
2. В правом верхнем углу может быть кнопка **"Deploy"**
3. Нажмите на неё → выберите **"Redeploy"**

## 📸 Где искать кнопку Redeploy:

### Вариант A: В списке деплоев
```
Vercel Dashboard
  └── Ваш проект
      └── Deployments (в меню слева)
          └── [Список деплоев]
              └── Последний деплой
                  └── "..." (три точки справа)
                      └── "Redeploy"
```

### Вариант B: На странице деплоя
```
Vercel Dashboard
  └── Ваш проект
      └── Deployments
          └── [Нажмите на деплой]
              └── Страница деплоя
                  └── Кнопка "Redeploy" (справа вверху)
```

### Вариант C: После изменения переменных
```
Vercel Dashboard
  └── Ваш проект
      └── Settings
          └── Environment Variables
              └── [Измените переменные]
                  └── "Save"
                      └── Уведомление "Redeploy to apply changes?"
                          └── "Redeploy"
```

## ⚡ Самый быстрый способ:

**Просто сделайте пустой коммит:**

```bash
git commit --allow-empty -m "Force redeploy"
git push
```

Vercel автоматически задеплоит!

## 🔍 Если кнопки нет:

1. **Проверьте, что вы вошли в правильный аккаунт**
2. **Проверьте, что у вас есть права на проект**
3. **Попробуйте обновить страницу (F5)**
4. **Используйте способ через Git** (самый надежный)

## ✅ После редеплоя:

1. Дождитесь завершения деплоя (статус "Ready")
2. Проверьте сайт - изменения должны примениться
3. Проверьте логи, если что-то не работает

---

**Совет:** Самый простой способ - через Git (пустой коммит). Это всегда работает!
