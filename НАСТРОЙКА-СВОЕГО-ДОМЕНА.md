# 🌐 Настройка своего домена seo-update.ru

## 📋 Текущая ситуация:

Vercel автоматически создал домены:
- `seo-updateru-git-main-shelpakovzhenya-arts-projects.vercel.app`
- `seo-update-160kfrl60-shelpakovzhenya-arts-projects.vercel.app`

**Это нормально!** Это автоматические домены Vercel. Теперь нужно добавить **свой домен** `seo-update.ru`.

---

## ✅ ШАГИ ДЛЯ НАСТРОЙКИ seo-update.ru:

### Шаг 1: Добавить домен в Vercel

1. В Vercel Dashboard (где вы сейчас) → найдите вкладку **"Domains"** или **"Settings"** → **"Domains"**
2. Нажмите кнопку **"Add Domain"** или **"Add"**
3. Введите: `seo-update.ru`
4. Нажмите **"Add"** или **"Continue"**

### Шаг 2: Vercel покажет DNS записи

После добавления домена Vercel покажет вам **DNS записи**, которые нужно добавить у регистратора домена.

**Обычно это:**
- **A запись**: `@` → IP адрес (например: `76.76.21.21`)
- **CNAME запись**: `www` → `cname.vercel-dns.com`

**Или:**
- **CNAME запись**: `@` → `cname.vercel-dns.com`
- **CNAME запись**: `www` → `cname.vercel-dns.com`

### Шаг 3: Добавить DNS записи у регистратора домена

1. Войдите в панель управления доменом (где покупали `seo-update.ru`)
2. Найдите раздел **"DNS"**, **"Управление DNS"** или **"DNS записи"**
3. Добавьте записи, которые показал Vercel:
   - Тип: `A` или `CNAME`
   - Имя: `@` (или оставьте пустым)
   - Значение: IP адрес или `cname.vercel-dns.com` (из Vercel)
   - TTL: `3600` или `Автоматически`
4. Для `www`:
   - Тип: `CNAME`
   - Имя: `www`
   - Значение: `cname.vercel-dns.com`
5. Сохраните изменения

### Шаг 4: Дождаться распространения DNS

- **Обычно:** 1-2 часа
- **Максимум:** 24-48 часов
- **Проверить можно:** https://dnschecker.org (введите `seo-update.ru`)

### Шаг 5: Обновить переменные окружения

1. В Vercel Dashboard → **Settings** → **Environment Variables**
2. Найдите или добавьте: `SITE_URL`
3. Установите значение: `https://seo-update.ru`
4. Нажмите **"Save"**
5. Появится уведомление → нажмите **"Redeploy"**

### Шаг 6: Проверить работу

После распространения DNS:
1. Откройте: `https://seo-update.ru`
2. Должен открыться ваш сайт
3. Проверьте SSL (должен быть зеленый замочек)
