# Студия Английского

SEO-friendly сайт и административная панель для онлайн-центра английского языка.

## Стек

- Next.js 14
- React 18
- Prisma
- SQLite по умолчанию
- Tailwind CSS

## Что реализовано

- публичная главная страница `Студии Английского`
- раздел `О нас` с акцентом на преимущества центра вместо персонального блока
- админка по мотивам архитектуры `seo-service-mvp`
- авторизация администратора
- CRUD-модули для:
  - программ
  - преподавателей
  - страниц
  - FAQ
  - отзывов
  - блога
  - лидов
  - настроек сайта

## Запуск

1. Создайте `.env` на основе `.env.example`
2. Установите зависимости:
   `npm install`
3. Подготовьте базу:
   `npm run setup`
4. Запустите проект:
   `npm run dev`

## Админка

- URL: `/admin/login`
- логин: берется из `ADMIN_USER`
- пароль: берется из `ADMIN_PASS`

## Структура

- `app/` — публичные страницы, админка и API
- `components/` — UI и админские компоненты
- `lib/` — Prisma, auth, конфиги ресурсов
- `prisma/` — схема и seed
- `public/` — публичные изображения
