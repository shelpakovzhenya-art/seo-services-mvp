# Студия Английского

Маркетинговый сайт для онлайн-центра английского языка с акцентом на личный бренд Лены, экспертную подачу и теплую образовательную атмосферу.

## Что есть сейчас

- стартовая главная страница с брендовой визуальной концепцией
- hero-блок с портретом основателя
- экспертный блок о Лене и ее преподавательском подходе
- блоки программ, методики, доверия, FAQ и заявки

## Структура

- `index.html` — главная страница
- `styles.css` — стили и визуальная система
- `script.js` — анимация появления секций и форма заявки
- `assets/images/` — фотографии основателя

## Запуск

Откройте `index.html` в браузере.
